マネーフォワード クラウド確定申告を申し込む前の確認事項

公開日 2026-06-25｜更新日 2026-06-25

この記事には広告リンクが含まれます。運営者は本サービスを未検証であり、公式情報と申込前の確認事項を整理しています。

マネーフォワード クラウド確定申告は、個人事業主やフリーランス向けのクラウド会計・確定申告サービスです。ここでは使用感を評価せず、2026年6月25日に公式サイトで確認できた情報を整理します。

公式サイトで確認できた主な機能

パーソナルミニとパーソナルの両プランで、青色・白色の確定申告書類作成、電子申告、銀行・クレジットカード明細の自動取込、メール・チャットによる操作サポートが案内されています。

  • 青色申告・白色申告の書類作成
  • 電子申告（e-Tax）への対応
  • 銀行・クレジットカード明細の自動取込
  • 請求書など関連サービスとの連携
  • メール・チャットによる操作サポート

料金とプランの違い

公式サイトでは、パーソナルミニが年払い換算で月900円、一括10,800円／年、パーソナルが年払い換算で月1,280円、一括15,360円／年と案内されています。いずれも税抜表示です。

パーソナルミニでは消費税申告を利用できません。課税事業者やインボイス制度への対応が必要な場合は、パーソナル以上の機能を確認する必要があります。

  • パーソナルミニ：必要最低限の確定申告機能を使いたい人向け
  • パーソナル：消費税申告や詳細なレポートも必要な人向け
  • 年払いと月払いでは総額が異なる
  • レシート撮影はプランごとに件数や追加料金の条件がある

無料トライアルで確認すること

公式サイトでは、クレジットカード登録不要の1カ月無料トライアルが案内されています。終了後に自動で有料プランへ移行せず、作成データは引き継がれるとされています。

  • 利用中の銀行・カードが連携対象か
  • 仕訳候補を自分で確認・修正できるか
  • 必要な帳票やCSVを出力できるか
  • スマートフォンから電子申告するための要件
  • 有料化する場合の年額と解約手順

申込前の注意点

料金や機能は変更される可能性があります。申込時には公式サイトで最新条件を確認してください。また、税務判断が必要な内容は会計ソフトだけで判断せず、税務署や税理士などの専門家へ確認する必要があります。

まとめ

候補に入れる場合は、まず無料トライアルで日常の取引を数件入力し、自分の口座連携、仕訳確認、帳票出力まで試してください。機能の多さより、毎月継続して入力できるかを判断基準にします。

広告

広告リンクです。最新の料金、無料期間、提供条件はリンク先でご確認ください。
